# Bundled backdrop posters — provenance & license

The `builtin-NN.jpg` files are **public-domain** vintage film posters
(silent-era / 1920s) sourced from Wikimedia Commons. Their copyright has
expired, so they are free to redistribute. Used as the login-screen
backdrop when no library art is available yet (fresh boot / offline).

Source categories:
- https://commons.wikimedia.org/wiki/Category:Film_posters_of_silent_films
- https://commons.wikimedia.org/wiki/Category:1920s_film_posters
